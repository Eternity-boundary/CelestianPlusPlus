# Celestian++
![Language](https://img.shields.io/badge/language-C++-red)

# Celestian的进阶版本，适用于一念成仙

本项目需要安装LLOneBOT才能使用！
请前往https://github.com/LLOneBot/LLOneBot

下载安装

## 使用方法
1.打开LLoneBOT配置页面

2.启用OneBot协议

3.HTTP监听端口设置为3000

4.启用HTTP心跳

5.启用HTTP上报并将上报地址设置为http://localhost:7777/

6.开始运行Celestian++
